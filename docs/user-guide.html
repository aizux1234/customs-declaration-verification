เอกสารคู่มือ — สำหรับการทดสอบระบบ (Prototype)

ระบบตรวจสอบข้อมูลใบขนสำหรับการปล่อยสินเชื่อ

Verify Export Risk Intelligent — คู่มือรหัสผู้ใช้งานและวิธีการใช้งาน
เวอร์ชัน 1.0 (Prototype)  |  ปรับปรุง 18 พฤษภาคม 2569

1. การเปิดใช้งานระบบ

ขั้นที่ 1 — เปิดโปรแกรมเทอร์มินัลที่โฟลเดอร์โปรเจค แล้วรันคำสั่ง:  npm run dev

ขั้นที่ 2 — รอจนระบบแสดงข้อความ ready พร้อมที่อยู่เว็บไซต์

ขั้นที่ 3 — เปิดเบราว์เซอร์ (แนะนำ Chrome หรือ Edge) แล้วเข้าที่:  http://localhost:5173

ขั้นที่ 4 — เข้าสู่ระบบด้วยรหัสผู้ใช้งานในตารางหัวข้อถัดไป

2. รหัสผู้ใช้งานสำหรับทดสอบ

รหัสผ่านของทุกบัญชีเหมือนกันทั้งหมด คือ   Passw0rd!
Username บทบาท (Role) สิทธิ์การใช้งาน
superadminผู้ดูแลระบบสูงสุดเข้าถึงได้ทุกเมนู — แนะนำใช้บัญชีนี้สำหรับทดสอบ
admin01ผู้ดูแลระบบจัดการผู้ใช้งานและบทบาท
officer01เจ้าหน้าที่สินเชื่อตรวจสอบใบขน, รายงานประวัติการค้นหา
officer02เจ้าหน้าที่สินเชื่อตรวจสอบใบขน, รายงานประวัติการค้นหา
bde01เจ้าหน้าที่บันทึกข้อมูลผู้กู้จัดการข้อมูลผู้กู้
cde01เจ้าหน้าที่บันทึกข้อมูลใบขนจัดการข้อมูลผู้กู้และการเชื่อมโยงใบขน
auditor01ผู้ตรวจสอบดูข้อมูลทุกส่วน (อ่านอย่างเดียว) + Activity Log
หมายเหตุ: ระบบไม่มีหน้าสมัครสมาชิก — บัญชีผู้ใช้งานทั้งหมดสร้างโดยผู้ดูแลระบบ หากกรอกรหัสผ่านผิดติดต่อกัน 5 ครั้ง บัญชีจะถูกล็อกอัตโนมัติ

3. การสลับบทบาท (Role Switcher)

หลังเข้าสู่ระบบแล้ว ที่ แถบด้านบนขวา จะมีช่องเลือก (Dropdown) สำหรับสลับบทบาท โดยไม่ต้องออกจากระบบ — เมื่อเลือกบทบาทใหม่ เมนูด้านซ้ายจะเปลี่ยนตามสิทธิ์ของบทบาทนั้นทันที (ฟังก์ชันนี้จัดทำไว้เพื่อความสะดวกในการสาธิตระบบ)

4. วิธีใช้งานเมนูหลัก

ตรวจสอบใบขน — กรอก Reference Number และเลขที่ใบขน แล้วกด “ค้นหา” ระบบจะแสดงข้อมูลใบขน รายการสินค้า และปุ่ม “Preview เอกสาร” เพื่อดูเอกสารใบขนแบบเต็ม  (ทดลองกรอกเลข TIMEOUT000000 เพื่อดูการแจ้งเตือนเมื่อระบบศุลกากรไม่ตอบสนอง)

รายงานประวัติการค้นหา — ดูประวัติการค้นหาใบขนทั้งหมด กรองตามช่วงวันที่/ผู้ค้นหา/เลขที่ใบขน และกดปุ่ม “Export Excel” เพื่อดาวน์โหลดเป็นไฟล์ .xlsx

จัดการผู้ใช้งาน — เพิ่ม / แก้ไข / ปิดใช้งานบัญชีผู้ใช้ และกำหนดบทบาท (เฉพาะผู้ดูแลระบบ)

จัดการข้อมูลผู้กู้ — เพิ่ม / แก้ไขข้อมูลผู้กู้ ดูรายละเอียด และเชื่อมโยงใบขนเข้ากับผู้กู้แต่ละราย

Activity Log — บันทึกทุกการกระทำในระบบ ดูได้อย่างเดียว แก้ไข/ลบไม่ได้ (เฉพาะผู้ดูแลระบบสูงสุดและผู้ตรวจสอบ)

5. ข้อควรทราบ

  • ระบบนี้เป็น ต้นแบบ (Prototype) — ข้อมูลทั้งหมดเป็นข้อมูลจำลอง ยังไม่ได้เชื่อมต่อ API กรมศุลกากรจริง
  • ข้อมูลเก็บอยู่ในหน่วยความจำชั่วคราว — เมื่อรีเฟรชหน้าเว็บ ข้อมูลจะกลับเป็นค่าเริ่มต้น
  • รองรับเบราว์เซอร์ Chrome, Edge, Firefox และ Safari รุ่นล่าสุด
  • ปิดเซิร์ฟเวอร์เมื่อเลิกใช้งานด้วยการกด Ctrl + C ในเทอร์มินัล
